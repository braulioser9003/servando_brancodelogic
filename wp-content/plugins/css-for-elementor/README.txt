== Description ==

11+ Elementor Addons and the very 1st Elementor EXTENDER that will provide you more custom options for Elementor Widgets and make Elementor more user friendly without having any custom CSS knowledge. Its pre-written codes will do the tricks for you. Before you might need to write custom CSS to do such things, but now you wont! 

游누 This plugin will add new 11 widgets (including Pro). Besides that this will enhance the current widgets of Elementor and Elementor Pro. 

游누 If you are not familiar with Custom CSS, but still you need some more features from Elementor widgets then this plugin is for you.

== Installation ==

1. Search for 'ElementsCSS for Elementor' in Dashboard -> Plugins -> Add New
2. Install and Activate
3. Enjoy
